## 웹 서버와 웹 어플리케이션 서버 
<pre>
클라이언트 = 브라우저
웹 서버 = 정적인 처리를 하는 공간 
웹 애플리케이션 = 소스코드(로직이 실행되는 공간) / 동적인 처리를 하는 공간
데이터베이스 = 정보를 가지고 있는 공간
</pre>
